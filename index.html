About


India in Pixels is a simple image gallery showcasing the diverse beauty of Indian cities. The project features photographs from places in India like Hyderabad, Mumbai, etc highlighting culture, architecture, and landscapes.
This website is part of my journey in learning web development.
It has been created using basic HTML and CSS, with a focus on clean layout, visual balance, and responsive design. Through this project, I aim to explore how visual storytelling can reflect the diversity of a country as vast as India.
